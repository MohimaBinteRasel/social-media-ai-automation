# Social Media AI Agent using n8n

This project is an AI-powered social media automation system built using **n8n**.  
It automatically collects content, generates professional posts using AI, and publishes them to platforms like **LinkedIn**, **Twitter**, and **Telegram**.

## Workflow Overview

1. Schedule trigger starts the workflow
2. Content is fetched from external sources
3. Duplicate items are filtered
4. AI generates platform-optimized content
5. Content is validated
6. Post is published automatically
7. Posting status is updated
